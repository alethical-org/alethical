# UI mockups (redesign reference)

High-fidelity visual mockups for the redesign, authored in Claude design and
exported as self-contained HTML (one file per screen). These are the **visual
aesthetic reference** for the token/primitives kit and the front-end UI build.

Scope note: treat these as the *aesthetic* source of truth (color, type, radii,
shadows, gradient, spacing) plus the real-intent structural bits (nav/IA,
background, feature placement). Content/copy and detailed UX are placeholder and
refined during the build. Navigation reconciles against `apps/frontend/src/navigation/ia.ts`
(the registry is source of truth for IA). Mockups are fixed-width desktop comps;
responsive/mobile behavior is derived during the build.

See `docs/mvp-redesign-plan.md` for the full plan.
